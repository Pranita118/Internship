
# Retail Sales Prediction using Machine Learning

This project predicts retail product sales using Machine Learning techniques on the BigMart Sales Dataset.

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Feature Engineering
* Random Forest Regression Model
* Sales Prediction & Evaluation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

[BigMart Sales Dataset](https://www.kaggle.com/datasets/brijbhushannanda1979/bigmart-sales-data?utm_source=chatgpt.com)
